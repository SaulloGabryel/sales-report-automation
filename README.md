# Sales Report Automation

Este projeto demonstra como automatizar relatórios de vendas usando Python, Excel e Matplotlib. Ele lê uma planilha de vendas, calcula métricas importantes, gera gráficos e atualiza a planilha automaticamente com os resultados.

## Funcionalidades

- Calcula o faturamento total e mensal
- Identifica os produtos mais vendidos
- Identifica os vendedores com melhor desempenho
- Gera gráficos automáticos em PNG
- Insere gráficos diretamente na planilha Excel
- (Opcional) Exporta PDF com resumo do relatório

## Estrutura do projeto

sales_report_project/
│
├── main.py # Script principal
├── data/ # Planilha de vendas original
│ └── planilha de vendas.xlsx
├── output/ # Gráficos e planilhas geradas
└── README.md

## Tecnologias

- Python (pandas, matplotlib, openpyxl)
- Excel (.xlsx)
- Matplotlib para visualização de dados
